hyalus.parse.factory module

Functionality for generating ResultsParser instances from file paths

hyalus.parse.factory._name_map: dict[str, Type[hyalus.parse.base.ResultsParser]] = {}

Controls the ResultsParser class that is associated with specific file names - takes priority over extension

hyalus.parse.factory._ext_map: dict[str, Type[hyalus.parse.base.ResultsParser]] = {'.csv': <class 'hyalus.parse.csv.CSVDataFrameParser'>, '.h5': <class 'hyalus.parse.h5.H5Parser'>, '.json': <class 'hyalus.parse.json.JSONParser'>, '.tsv': <class 'hyalus.parse.tsv.TSVDataFrameParser'>}

Controls the ResultsParser class that is associated with certain extensions

hyalus.parse.factory.get_parser(path, name_map={}, ext_map={'.csv': <class 'hyalus.parse.csv.CSVDataFrameParser'>, '.h5': <class 'hyalus.parse.h5.H5Parser'>, '.json': <class 'hyalus.parse.json.JSONParser'>, '.tsv': <class 'hyalus.parse.tsv.TSVDataFrameParser'>})

Based on a given path’s filename/extension, determine which applicable ResultsParser, if any, should parse it and create an instance of that parser

Parameters:

  • path (str | Path) – The path corresponding to the file to parse

  • name_map (dict[str, Type[ResultsParser]], default: {}) – The mapping of file name to ResultsParser class to use, defaulting to _name_map

  • ext_map (dict[str, Type[ResultsParser]], default: {'.json': <class 'hyalus.parse.json.JSONParser'>, '.csv': <class 'hyalus.parse.csv.CSVDataFrameParser'>, '.tsv': <class 'hyalus.parse.tsv.TSVDataFrameParser'>, '.h5': <class 'hyalus.parse.h5.H5Parser'>}) – The mapping of file extension to ResultsParser class to use, defaulting to _ext_map

Returns:

Optional[ResultsParser] – The instantiated ResultsParser or None if no corresponding parser could be found
