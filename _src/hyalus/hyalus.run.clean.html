hyalus.run.clean module

Hyalus test run cleanup

class hyalus.run.clean.HyalusCleanRunner(runs_dir, to_clean=None, tags=None, tag_op=<built-in function all>, oldest=None, newest=None, force=False)

Bases: object

Orchestration for hyalus test run cleanup

Ctor.

Parameters:

  • runs_dir (str | Path) – The runs directory to clean up

  • to_clean (Optional[Sequence[str]], default: None) – List of test names/test plans corresponding to tests to clean up. If none given, ALL test runs will be fair game

  • tags (Optional[Sequence[str]], default: None) – Tags that when matched will mark a test for removal. If none given, ALL given tests will be removed

  • tag_op (Callable[..., bool], default: <built-in function all>) – Operator to apply to tag matching - all if test must have all given tags, any if the test must have any of the given tags

  • oldest (Optional[date], default: None) – Date string in format YYYY-MM-DD marking the oldest date a test can have and be kept. Any tests older than this date will be removed. If none given, the earliest possible date is used.

  • newest (Optional[date], default: None) – Date string in format YYYY-MM-DD marking the newest date a test can have and be kept. Any tests newer than this date will be removed. If none given, the newest/latest possible date is used.

  • force (bool, default: False) – If set to True, will force remove test runs matching criteria. If set to False, will prompt user confirmation prior to actual removal of tests
