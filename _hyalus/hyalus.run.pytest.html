hyalus.run.pytest.applydecorator(decorator)

Decorator that applies a given decorator to all defined methods of a class. For example:

@applydecorator(staticmethod)
class MyClass:

    @pytest.fixture(name="my_fixture")
    def fixture_my_fixture():
        ...

    def method_1():
        ...

    def method_2():
        ...

is roughly equivalent to

class MyClass:

    @staticmethod
    @pytest.fixture(name="my_fixture")
    def fixture_my_fixture():
        ...

    @staticmethod
    def method_1():
        ...

    @staticmethod
    def method_2():
        ...

This allows you to not have to manually decorate methods with staticmethod() or add self as a method argument for pytest-style test classes. Your linter with probably complain (pylint E0211 and E0213) but at runtime there won’t be any issues.

Parameters

decorator (Callable[[Callable], Callable]) – The decorator to decorate all methods with

Returns

Callable[[type], type] – Function that decorates a given class’ methods with the given decorator

hyalus.run.pytest.run_steps(*steps, use_run_dir=True)

Decorator that will create a temp directory if needed and run given Steps prior to execution of a function. For example:

@run_steps(ExampleStep(1, 2), ExampleStep(3, 4), use_run_dir=False)
def three_input_sum(a, b, c):
    return a + b + c

would end up as a function defined as:

def three_input_sum(a, b, c):
    ExampleStep(1, 2).run(1, <some_temp_dir_path>)
    ExampleStep(3, 4).run(2, <some_temp_dir_path>)
    return a + b + c

When used in the context of a pytest-style test, the run_dir fixture (defined in this module) may be used instead of an arbitrary temp directory:

@run_steps(ExampleStep(1, 2), ExampleStep(3, 4), use_run_dir=True))
def test_something():
    my_condition = some_module.function_call(True, False, True)

    assert my_condition

would end up as (note the addition of run_dir as a function parameter):

def test_something(run_dir):
    ExampleStep(1, 2).run(1, <some_temp_dir_path>)
    ExampleStep(3, 4).run(2, <some_temp_dir_path>)
    my_condition = some_module.function_call(True, False, True)

    assert my_condition

run_steps() can also be used with the applydecorator() class decorator to apply the given Step execution prior to all defined tests under a test class with a single line of code.

Parameters

  • *steps – Steps to run prior to function execution. Steps must define a __str__ method that returns a string representing syntactically what would have to be written to initialize the Step

  • use_run_dir – Use the run_dir fixture as opposed to dynamically creating a temp directory for the Step to run in

Returns

Callable[[Callable], Callable] – The function decorator
